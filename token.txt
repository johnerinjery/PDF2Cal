token.json
==========

This file is automatically generated after the user logs into Google for the first time while running the application.

It stores the authenticated session/token so the user does not need to log in every time.

How It Is Created
-----------------

1. Run the program:
   python src/main.py

2. A browser window will open asking for Google login permission.

3. After successful login and consent,
   token.json will be created automatically in the project directory.


Important
---------
Do NOT share token.json.

This file grants access to your Google Calendar account.

If compromised:
- delete token.json
- revoke app access from your Google Account security settings
